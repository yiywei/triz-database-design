# TRIZ Database Design

The purpose of this repository is to translate the database design into SQL queries and Django code for the TRIZ Innovation Capstone Project.

## Tech Stack
```
- Python 3.13 
- Django 5.1.7 
- MySQLClient 2.2.7
- MySQL 8.0 or higher
- SQL Backend Development
```
## Installation
```
- triz 
- triz_django
```
